# AI Workflow

## Purpose

Defines how AI agents should work in this repository.

## Workflow

- Read the relevant Brain files before making changes.
- Keep implementation scoped to the active task or handoff.
- Update Brain docs when architecture, product behavior, tasks, or durable decisions change.
- Prefer small, explicit changes over broad rewrites.
- Run the narrowest useful validation before finishing.

## Completion Notes

Final responses must include changed Brain files or state that no Brain documentation update was required with a reason.
