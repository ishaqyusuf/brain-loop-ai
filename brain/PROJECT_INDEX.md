# Project Index

## Purpose

Maps the repository structure for humans and AI agents.

## Repository Map

- `apps/desktop`: Tauri desktop app.
- `apps/desktop/src`: React UI.
- `apps/desktop/src-tauri`: Rust desktop shell and automation commands.
- `packages/brain-core`: shared Brain state types and constants.
- `packages/desktop-client`: frontend wrapper around Tauri commands/events.
- `brain`: project memory and workflow docs.
